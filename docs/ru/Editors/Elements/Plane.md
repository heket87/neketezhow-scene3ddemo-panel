# plane

Плоскость — это плоский прямоугольный объект. Она хорошо подходит для пола, стен, подложек, фонов, экранов и панелей.

## Когда использовать

- пол или потолок;
- стену;
- экран или табло;
- фон для текста;
- подложку под другие объекты.

## Основные параметры

### Size settings

Обычно плоскость настраивается по ширине и высоте. В зависимости от вашей сцены важны прежде всего размеры по двум осям.

### Render on both sides

Если включено, плоскость видна с обеих сторон.

Это особенно полезно, когда:

- пользователь может обойти объект сзади;
- плоскость используется как свободно висящая панель;
- объект может быть повернут так, что обратная сторона тоже должна отображаться.

Если параметр выключен, обратная сторона может быть невидимой.

## Что ещё доступно

Плоскость поддерживает общие настройки:

- прозрачность;
- цвет;
- границы;
- позицию;
- поворот;
- ссылки;
- текстовое окно;
- анимации.

Подробности: [ElementsOverview.md](ElementsOverview.md)

## Практический совет

Для фонов и больших поверхностей чаще всего удобно:

- включить двустороннее отображение;
- подобрать статический цвет;
- не перегружать плоскость лишними эффектами.
