# Line3D

Line3D — это пространственная линия или трубка, построенная по набору точек или по последовательности уже существующих элементов.

Такой объект удобно использовать как маршрут, кабель, трубу, соединение или траекторию движения.

## Когда использовать

- кабельные трассы;
- трубопроводы;
- соединительные линии;
- маршруты перемещения;
- технические связи между объектами.

## Основные параметры

### Tube radius query

Источник радиуса линии.

Можно выбрать:

- **static** — радиус задаётся вручную;
- поле данных — радиус будет приходить из запроса.

### Tube radius

Толщина линии в режиме **static**.

### Tubular segments (along path)

Количество сегментов вдоль линии.

Больше сегментов:

- линия выглядит плавнее;
- нагрузка выше.

### Radial segments (around tube)

Количество сегментов вокруг сечения линии.

### Smooth curve (CatmullRom)

Сглаживает путь между точками.

Если выключить параметр, траектория будет более ломаной.

### Closed loop

Замыкает линию в кольцо.

### Set points manually

Выбор режима построения.

#### Включено

Путь задаётся вручную через список точек.

#### Выключено

Путь строится по списку связанных элементов.

## Режим ручных точек

### Point coordinates

Для каждой точки задаются:

- **X**;
- **Y**;
- **Z**.

Точки можно:

- добавлять;
- удалять;
- менять их порядок.

### Data (threshold / gradient color for segment after this point)

Можно задать поле данных для сегмента, который идёт после этой точки.

Это полезно, если вы хотите, чтобы разные участки линии имели разную окраску или поведение.

### Use gradient color

Включает окраску сегмента по общему градиенту панели.

## Режим Connected elements

В этом режиме линия строится по уже существующим объектам.

### Element to connect

Выбор элемента в списке.

Элементы можно:

- переставлять вверх и вниз;
- удалять;
- добавлять новые.

Чтобы построить осмысленную линию, обычно нужно не меньше двух элементов.

## Что ещё доступно

Line3D поддерживает:

- цвет и градиент;
- двусторонний рендер;
- видимость;
- позицию;
- тени;
- ссылки;
- текстовые окна;
- анимации.

## Особое применение

Line3D часто используется как путь в **Movement animations**. Это позволяет заставить объект двигаться по уже готовой линии.

## Практический совет

Если линия должна быть маршрутом движения, сначала настройте сам путь, а уже потом подключайте к нему анимацию перемещения.
