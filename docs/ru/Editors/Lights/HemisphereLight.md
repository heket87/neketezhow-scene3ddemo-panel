# Hemisphere Light

Мягкий свет типа «небо сверху, земля снизу». Он хорошо подходит для наружных сцен и для деликатного заполняющего освещения.

## Основные параметры

### Light type

Тип источника света. Для этого описания должен быть выбран соответствующий вариант.

### Light name

Имя источника в списке. Удобно использовать понятные названия, например `Main light`, `Back light`, `Lamp left`.

### Sky color

Цвет света сверху.

### Ground color

Цвет света снизу.

### Add Light Helper

Помогает визуально понять ориентацию источника.

### Light intensity

Яркость света. Небольшие значения обычно подходят для мягкого освещения, большие — для акцентного.

### Light color

Цвет света. С его помощью можно сделать сцену теплее, холоднее или создать визуальное различие между несколькими источниками.

## Когда использовать

- для основного освещения сцены;
- для подсветки отдельных объектов;
- для создания нужного настроения и читаемости формы.

## Практические советы

### Если сцена слишком плоская

Добавьте второй источник света с другой стороны или комбинируйте несколько типов света.

### Если тени слишком тяжёлые для браузера

Выключите **Dynamic shadows** у этого источника или глобально у панели.

### Если непонятно, куда светит источник

Включите **Add Light Helper** и подберите положение уже по маркеру в сцене.
