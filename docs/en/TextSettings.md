# Text settings

Этот раздел описывает текстовые окна, которые можно привязать к элементу или к отдельной части модели. Такие окна используются для подписей, значений из Grafana, ссылок, технических подсказок и пояснений.

Текстовое окно отличается от самого 3D-текста.

- **text2d** и **text3d** — это отдельные типы элементов сцены.
- **Text settings** — это всплывающее окно, привязанное к уже существующему объекту.

## Где это используется

Текстовое окно можно включить:

- у обычного элемента через **Add text window**;
- у части разобранной модели через аналогичный переключатель в настройках этой части.

После включения появляется блок **Text settings**.

## Глобальные и локальные настройки

### Use global text settings

Если параметр включён, окно наследует оформление из общего раздела **Global text settings**.

Это удобно, когда вы хотите, чтобы все подписи в панели выглядели одинаково.

Если параметр выключен, вы можете настроить это окно отдельно: свой цвет текста, рамку, линию-выноску, ширину и фон.

## Содержимое текста

### Element text

Основное содержимое окна.

Поддерживаются:

- обычный текст;
- значения из полей данных;
- переменные Grafana;
- простые вычисления;
- форматирование чисел.

## Подстановки данных

### Подстановка значения поля

Используйте запись:

```text
{ИмяПоля}
```

Пример:

```text
Температура: {Value}
```

### Переменные Grafana

Используйте запись:

```text
$имя_переменной
```

Пример:

```text
Стойка: $rack
```

### Арифметика

Внутри фигурных скобок можно делать простые операции:

- `+`
- `-`
- `*`
- `/`
- `^`
- `%`

Пример:

```text
Мощность: {Value * 1000}
```

### Форматирование числа

Можно округлять вывод:

```text
{Value:.0f}
{Value:.1f}
{Value:.2f}
```

Примеры:

```text
{Value:.0f}
{Value / 1000:.2f}
{Value * 100:.1f}
```

## Положение окна

### Use real visual center for text position

Если включено, позиция текстового окна считается от визуального центра объекта, а не просто от его условной точки привязки.

Это особенно полезно для:

- сложных моделей;
- вытянутых объектов;
- объектов с несимметричной формой.

### Text X offset relative to object
### Text Y offset relative to object
### Text Z offset relative to object

Смещение окна относительно объекта по трём осям.

Используйте эти поля, если окно:

- перекрывает сам объект;
- стоит слишком близко;
- должно находиться сбоку или выше объекта.

## Ссылки в текстовом окне

### Make text window clickable

Делает текстовое окно кликабельным.

### Link URL

Адрес, который откроется по клику.

В этом поле тоже можно использовать подстановки данных и переменные Grafana.

### Open in current window

Если включено, ссылка откроется в текущей вкладке.

Если выключено, обычно открывается новая вкладка.

## Управление видимостью текстового окна

Текстовое окно можно показывать всегда, по наведению или по условию из данных.

### Data field for visibility

Выбор поля данных, которое управляет показом окна.

Есть специальный вариант **static** — в этом режиме окно не зависит от данных.

### Visibility condition mode

Режим проверки условия.

#### Numeric comparison

Подходит для числовых данных.

#### Regex

Подходит для строк и текстовых значений.

### Hide when value is

Оператор сравнения для числового режима.

Примеры:

- меньше;
- меньше или равно;
- больше;
- больше или равно;
- равно;
- не равно.

### Threshold value

Порог, с которым сравнивается значение.

### Regex — text hidden when matches

Регулярное выражение. Если значение соответствует шаблону, окно скрывается.

### Static text visibility

Используется, когда выбран режим **static**. Позволяет явно включить или выключить окно без привязки к данным.

## Локальное оформление окна

Эти настройки доступны, когда **Use global text settings** выключен.

### Show leader line

Показывает линию от объекта к текстовому окну.

### Custom leader line color

Разрешает использовать отдельный цвет для линии.

### Leader line color

Цвет линии.

### Leader line width (px)

Толщина линии в пикселях.

### Show text statically

Текст виден постоянно.

### Show text if data condition match

Если постоянный показ выключен, можно показывать текст только при выполнении условия по данным.

### Size (px)

Размер шрифта.

### Maximum text window width (px)

Максимальная ширина окна.

### Text color

Цвет текста.

### Border color

Цвет рамки.

### Border thickness (px)

Толщина рамки.

### Text window background color

Фоновый цвет окна.

## Цвет фона окна по данным

### Use data for background color

Позволяет окрашивать фон окна в зависимости от значения из данных.

### Data source for background color

Поле данных, которое будет использоваться для выбора цвета.

### Use gradient for background color

Включает окраску через общий градиент панели.

## Разница между текстовым окном и вспомогательной информацией

### Text settings

Это подпись, привязанная к конкретному объекту.

### Add auxiliary information

Это отдельная информационная панель, которая появляется в углу всей панели при наведении на объект.

Если вам нужна именно подпись возле объекта, используйте **Add text window**.

Если нужна дополнительная служебная панель в углу, используйте **Add auxiliary information**.

## Практические советы

### Для коротких подписей

Обычно достаточно:

- небольшого смещения по Y;
- постоянного показа;
- одной строки текста.

### Для значений из Grafana

Рекомендуется:

- использовать форматирование числа;
- ограничивать ширину окна;
- делать фон полупрозрачным.

### Для кликабельных подсказок

Удобно включить:

- кликабельность;
- открытие в новой вкладке;
- явный контрастный фон.
