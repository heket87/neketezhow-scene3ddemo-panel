# Custom Element

Пользовательский элемент позволяет создать форму по собственному контуру. Вы сами задаёте вершины, а затем можете выдавить форму в 3D.

Это один из самых гибких типов объектов.

## Когда использовать

- нестандартные фигуры;
- логотипы и пиктограммы;
- произвольные контуры;
- формы, которых нет среди готовых примитивов.

## Основные параметры

### Extrusion settings

Определяет, вдоль какой оси форма будет выдавливаться.

Доступны варианты:

- **Extrude along X axis**;
- **Extrude along Y axis**;
- **Extrude along Z axis**.

Выбор оси влияет на то, в какой плоскости лежит исходный контур и куда будет направлена толщина объекта.

### Point coordinates

Список точек контура.

Для каждой точки задаются:

- **X coordinate**;
- **Y coordinate**.

Также точки можно:

- перемещать вверх и вниз по списку;
- удалять;
- добавлять новые.

Порядок точек очень важен: именно он определяет форму границы.

### Bevel Enabled

Включает фаску на краях формы.

### Settings bevel

#### steps

Количество шагов по глубине выдавливания.

#### Bevel Thickness

Глубина фаски.

#### Bevel Size

Размер скоса по краю.

#### bevelOffset

Смещение фаски.

#### bevelsegments

Количество сегментов фаски.

## Что ещё доступно

Пользовательский элемент поддерживает общие свойства:

- цвет;
- прозрачность;
- позицию;
- поворот;
- границы;
- ссылки;
- текстовые окна;
- анимации.

Подробности: [ElementsOverview.md](ElementsOverview.md)

## Практические советы

### Для простых форм

Используйте минимально необходимое количество точек. Так форму будет проще редактировать.

### Для аккуратного результата

Следите за порядком вершин. Если точки идут не по контуру, форма может выглядеть неожиданно.
