# cube

Куб — это базовый прямоугольный объёмный объект. Он подходит для большинства простых задач: корпуса, блоки, стойки, условные объекты, маркеры и строительные элементы сцены.

## Когда использовать

- коробки и контейнеры;
- условные блоки оборудования;
- стены, колонны, тумбы и постаменты;
- заготовки для более сложных композиций.

## Основные параметры

### Size settings

У куба обычно используются три размера:

- **Size X** — ширина;
- **Size Y** — высота;
- **Size Z** — глубина.

Каждый из размеров можно:

- задать вручную;
- связать с данными, если это нужно для динамики.

## Что ещё доступно

Куб поддерживает почти все общие настройки элемента:

- видимость;
- цвет и градиент;
- прозрачность;
- тени;
- позицию;
- поворот;
- границы;
- кликабельность;
- текстовое окно;
- вспомогательную информацию;
- анимации.

Подробности по общим настройкам: [ElementsOverview.md](ElementsOverview.md)

## Практический совет

Если вы только начинаете собирать сцену, куб — лучший первый тип элемента. На нём удобно проверить:

- свет;
- тени;
- привязку размеров к данным;
- базовые анимации.
