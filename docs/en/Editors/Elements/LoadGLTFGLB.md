# load gltf/glb

GLTF / GLB — один из самых удобных форматов для веб-3D. Он хорошо подходит для современных сцен, обычно быстрее и удобнее в использовании, чем многие старые форматы.

## Когда использовать

- если вы готовите модель специально для веба;
- если нужны встроенные анимации;
- если важен хороший баланс между качеством и размером файла.

## Основные особенности

### Поддержка анимаций

Для этого типа доступен параметр **Use object animation**. Если в файле есть готовые анимационные клипы, их можно использовать прямо в панели.

### Автоматическое определение размеров

После применения модель обычно можно сразу использовать без ручного ввода габаритов.

## Практический совет

Если у вас есть выбор формата для новой веб-сцены, чаще всего стоит начинать именно с GLTF / GLB.
