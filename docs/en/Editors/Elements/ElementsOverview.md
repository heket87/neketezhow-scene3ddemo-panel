# Elements Configuration

Этот раздел — сердце всей сцены. Именно здесь вы добавляете, удаляете и настраиваете объекты.

В одном списке могут одновременно находиться:

- простые геометрические фигуры;
- текст;
- линии;
- 3D-модели;
- объекты, созданные по изображению.

## Работа со списком элементов

### Add element

Добавляет новый объект в сцену.

### Имя элемента

Задаётся через **Element name**.

Имя нужно прежде всего для удобства:

- ориентироваться в списке;
- выбирать объекты в привязках;
- легче понимать структуру сцены.

### Copy

Создаёт копию элемента.

Это удобно, если нужно быстро размножить одинаковые объекты с небольшими отличиями.

### Delete

Удаляет элемент из сцены.

## Общие настройки элемента

Ниже перечислены параметры, которые встречаются у многих типов объектов.

## Button for editing

Позволяет назначить клавишу для быстрого редактирования элемента прямо через сцену.

### Key to edit element

Кнопка, по которой открывается редактирование выбранного объекта.

Это вспомогательная функция для более быстрого рабочего процесса.

## Element name

Понятное имя элемента в списке.

## Element type

Тип объекта. От выбора типа зависит, какие дополнительные блоки появятся ниже.

## Visibility

### Change element visibility

Включает отдельные настройки видимости.

### Visibility settings

#### Data for visibility

Поле данных, которое управляет показом объекта.

Есть специальный вариант **static** — объект не зависит от данных.

#### Visibility condition mode

Выбор режима проверки:

- числовое сравнение;
- регулярное выражение.

#### Hide when value is

Оператор сравнения для числового режима.

#### Threshold value

Порог для числового режима.

#### Regex when object is not visible

Шаблон для строкового режима.

#### Static element visibility

Явное включение или скрытие объекта без данных.

## Element transparency

Прозрачность объекта.

Общий смысл:

- `1` — полностью непрозрачный;
- `0` — полностью прозрачный.

Прозрачность полезна для стекла, подсветки, полупрозрачных областей и служебных объектов.

## Color settings

### Use query for element color

Позволяет брать цвет из данных.

### Data for element color

Выбор поля данных для окраски.

### Use gradient for element color

Включает окраску по общему градиенту панели.

По-умолчанию выключен и используются цвета thresholds если активирован Use query for element color


### Element color

Обычный статический цвет объекта.

### Center lines color

Специальный параметр для сетки. Управляет цветом центральных линий.

## Cast shadows

Объект отбрасывает тень на другие поверхности.

## Receive shadows

Объект принимает тени от других объектов.

## Position settings

Позволяют задать положение вручную или привязать его к другому объекту.

### Position mode

#### Manual coordinates

Обычное ручное задание координат.

#### Reference existing element / subelement

Положение рассчитывается относительно другого элемента или части модели.

Это удобно для сборки сцены из связанных объектов.

### Reference target

Объект, относительно которого будет рассчитана позиция.

### X offset / Y offset / Z offset

Смещение относительно выбранной цели.

### Resolved world position

Предварительный просмотр получившихся мировых координат.

### X coordinate / Y coordinate / Z coordinate

Ручные координаты объекта.

## Size settings

Размеры зависят от типа элемента.

Для простых фигур это обычно размер по осям. Для сферы — радиус и сегменты. Для текста — размер шрифта и глубина. Для сетки — число ячеек.

Подробности по типам смотрите в отдельных страницах.

## Rotation settings

### Rotation around X axis
### Rotation around Y axis
### Rotation around Z axis

Поворот объекта по трём осям.

## Render on both sides

Делает материал двусторонним.

Полезно для плоскостей, текста и тонких объектов, которые должны быть видны с обеих сторон.

## Add object borders

Добавляет обводку объекта.

### Border settings

- **Element border color** — цвет границы;
- **Element border thickness** — толщина границы.

## Make object clickable

Позволяет открыть ссылку по клику на объект.

### Link settings

- **Enter link** — адрес перехода;
- **Open in current window** — открывать в текущей вкладке.

## Add text window

Включает всплывающее текстовое окно, привязанное к объекту.

Полная документация: [TextSettings.md](../../TextSettings.md)

## Add auxiliary information

Включает дополнительную информационную панель при наведении.

Это не то же самое, что текстовое окно возле объекта. Вспомогательная информация обычно выводится в углу панели.

## Анимации

У элемента можно включить:

- **Enable axis rotation animations**;
- **Enable circular rotation animations**;
- **Enable movement animations**;
- **Enable color change animations**;
- **Use object animation** — для моделей с готовыми анимациями.

Подробности: [AnimationsOverview](../../Animations/AnimationsOverview.md)

## Практические советы

### Для чистой структуры сцены

- давайте элементам понятные имена;
- используйте копирование вместо ручного повторного создания.

### Для работы с данными

Сначала настройте объект в статическом виде, а затем постепенно привязывайте его параметры к данным.

### Для тяжёлых сцен

Не включайте всё сразу:

- тени;
- прозрачность;
- сложные градиенты;
- анимации;
- тяжёлые модели.

Проверяйте результат по шагам.
