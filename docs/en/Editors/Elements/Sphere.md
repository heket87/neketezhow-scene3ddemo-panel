# sphere

Сфера — это круглый объёмный объект. Её удобно использовать как маркер, индикатор, шарообразный объект, лампу, точку интереса или декоративный элемент.

## Когда использовать

- точечные маркеры;
- индикаторы состояния;
- шарообразные детали;
- визуальные якоря в сцене.

## Основные параметры

### Use query for radius

Позволяет получать радиус сферы из данных.

### Sphere radius

Базовый радиус сферы.

### Segments width
### Segments height

Количество сегментов по ширине и высоте.

Меньше сегментов:

- легче для браузера;
- форма грубее.

Больше сегментов:

- сфера выглядит более гладкой;
- выше нагрузка на рендеринг.

### Size X / Size Y / Size Z

Помимо радиуса у сферы могут использоваться размеры по осям. Это позволяет не только масштабировать сферу, но и превращать её в вытянутую форму, похожую на эллипсоид.

## Что ещё доступно

Сфера поддерживает:

- цвет;
- прозрачность;
- тени;
- позицию;
- поворот;
- текстовое окно;
- ссылки;
- анимации.

Подробности: [ElementsOverview.md](ElementsOverview.md)

## Практический совет

Если сфера нужна просто как индикатор, не делайте слишком большое количество сегментов. Для большинства служебных задач хватает умеренной детализации.
