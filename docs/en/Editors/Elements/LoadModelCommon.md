# Общие настройки загружаемых моделей

Эта страница описывает параметры, которые общие для моделей разных форматов:

- GLTF / GLB;
- FBX;
- OBJ;
- STL.

## Основной принцип работы

1. Вы выбираете тип модели.
2. Указываете адрес файла в **Enter URL of object**.
3. Применяете модель.
4. При необходимости разбираете её на части.
5. Настраиваете подсветку, материалы, текстовые окна и другое поведение.

## Model settings

### Enter URL of object

Адрес файла модели.

После применения модель загружается в сцену.

Для некоторых форматов при применении могут автоматически определяться габариты модели.

### Enable centering

Центрирует модель.

Это полезно, если исходный файл экспортирован не вокруг удобного центра и модель оказывается смещённой относительно точки привязки.

### Decompose object

Разбирает модель на отдельные части.

После этого появляется раздел **Subelements**, где можно настраивать каждую часть отдельно.

Это особенно полезно, если нужно:

- красить разные части модели по-разному;
- делать отдельные части кликабельными;
- задавать подписи и текстовые окна только для конкретных деталей;
- управлять прозрачностью отдельных частей.

Подробности: [Subelements.md](Subelements.md)

### Highlight elements on hover

Если модель разобрана на части, этот параметр включает подсветку части при наведении.

### Highlight color

Цвет подсветки при наведении.

## Что важно учитывать

### Модели могут быть тяжёлыми

Большое количество деталей и материалов заметно влияет на производительность.

### Разбор на части полезен, но тоже добавляет сложности

Если вам не нужно управлять отдельными частями модели, можно не включать **Decompose object**.

### Прозрачность и цвет у моделей часто настраиваются иначе

Для простых фигур удобно управлять материалом целиком, а у сложных моделей чаще приходится работать через части модели.

## Практический порядок работы

1. Сначала загрузите модель и проверьте её положение.
2. При необходимости включите центровку.
3. Потом решите, нужен ли разбор на части.
4. Уже после этого настраивайте цвет, подписи, подсветку и поведение отдельных элементов.
