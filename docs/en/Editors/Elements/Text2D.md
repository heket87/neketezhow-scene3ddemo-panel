# text2d

2D-текст — это плоский текстовый элемент. Он хорошо подходит для подписей, крупных заголовков, интерфейсных надписей и пояснений, которые должны читаться как обычный текст на панели.

## Когда использовать

- заголовки и подписи;
- краткие пояснения;
- текстовые блоки на плоскости;
- интерфейсные надписи внутри сцены.

## Основные параметры

### Font

Название шрифта.

### Alignment

Выравнивание текста, например по левому краю, по центру или по правому краю.

### Element text

Содержимое текста.

Поддерживаются:

- обычный текст;
- значения полей данных;
- переменные Grafana;
- форматирование и вычисления.

Подробности по подстановкам: [TextSettings.md](../../TextSettings.md)

### Smoothing

Сглаживание текста.

### Enter line height with px

Межстрочный интервал в пикселях.

### Enter letter spacing with px

Интервал между символами в пикселях.

### Window size

Размер текстовой области.

### Size

Размер шрифта в пикселях.

### Bevel Enabled

Параметр фаски присутствует и у этого типа, но для 2D-текста его визуальный эффект обычно менее заметен, чем у объёмного текста.

## Что ещё доступно

2D-текст можно:

- перемещать;
- поворачивать;
- окрашивать;
- делать кликабельным;
- сопровождать всплывающим текстовым окном;
- анимировать.

## Практический совет

Если нужен максимально читаемый текст, используйте именно **text2d**. Он обычно лучше подходит для надписей интерфейсного типа, чем объёмный 3D-текст.
