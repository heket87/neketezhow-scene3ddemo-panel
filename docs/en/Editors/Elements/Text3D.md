# text3d

3D-текст — это объёмные буквы внутри сцены. Такой текст выглядит как настоящий объект и хорошо подходит для крупных названий, маркировки зон, декоративных надписей и акцентных элементов.

## Когда использовать

- крупные заголовки в сцене;
- подписи зон и помещений;
- декоративные надписи;
- фирменные или презентационные элементы.

## Основные параметры

### JSON font URL

Путь к файлу шрифта в формате JSON.

Для объёмного текста нужен именно такой формат шрифта.

### Element text

Содержимое надписи.

Можно использовать как обычный текст, так и подстановки данных.

### Smoothing

Сглаживание текста.

### Enter line height with px

Межстрочный интервал.

### Enter letter spacing with px

Интервал между буквами.

### Use query for font size / Font size

Размер шрифта.

Можно задать вручную или получить из данных.

### Use query for Text depth / Text depth

Глубина выдавливания текста.

Чем больше значение, тем массивнее и объёмнее выглядят буквы.

### Render on both sides

Позволяет видеть текст с обеих сторон.

### Bevel Enabled

Включает фаску на краях букв.

Это делает текст более аккуратным и визуально дорогим.

### Settings bevel

#### steps

Количество шагов фаски.

#### Bevel Thickness

Глубина фаски.

#### Bevel Size

Ширина скоса по краю.

#### bevelOffset

Смещение фаски.

#### bevelsegments

Количество сегментов фаски для более плавного края.

## Что ещё доступно

3D-текст поддерживает общие свойства элемента:

- цвет;
- прозрачность;
- тени;
- позицию;
- поворот;
- ссылки;
- анимации.

Подробности: [ElementsOverview.md](ElementsOverview.md)

## Практический совет

Если текст должен быть легко читаем с дальнего расстояния и не нагружать сцену, не делайте слишком большую глубину и слишком сложную фаску.
