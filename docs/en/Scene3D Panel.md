# Scene3D Panel Plugin

Это главный раздел настроек панели. Здесь вы управляете содержимым сцены: объектами, камерами, освещением, текстовыми окнами, списком быстрых ссылок и общей цветовой шкалой.

Ниже разобраны все крупные блоки этого раздела.

## Elements Configuration

Это основной список объектов сцены.

Что можно делать в этом блоке:

- добавлять новые элементы кнопкой **Add element**;
- раскрывать настройки каждого элемента по имени;
- копировать элемент;
- удалять элемент;
- менять имя элемента для удобства навигации.

Подробная документация по типам объектов и их настройкам находится здесь:

- [Обзор элементов](Editors/Elements/ElementsOverview.md)
- [Анимации элементов](Animations/AnimationsOverview.md)
- [Текстовые окна элементов](TextSettings.md)

## Add fast links list

Включает список быстрых ссылок поверх сцены. Это полезно, если вы хотите добавить меню переходов прямо внутри панели.

После включения становятся доступны дополнительные параметры.

### Fast links list position

Положение списка ссылок внутри панели.

Обычно доступны четыре угла:

- верхний левый;
- верхний правый;
- нижний левый;
- нижний правый.

### Fast links list size

Размер блока со ссылками. Подходит для быстрой адаптации списка под маленькую или большую панель.

### Fast links settings

Здесь редактируется содержимое списка ссылок.

Для каждой ссылки доступны:

- **Link name** — подпись ссылки в списке;
- **Link URL** — адрес перехода;
- **Open in current window** — открывать ссылку в текущей вкладке, а не в новой.

Также ссылки можно:

- добавлять;
- копировать;
- удалять;
- перемещать вверх и вниз, чтобы менять порядок.

### Оформление списка ссылок

Цвета и рамки блока быстрых ссылок настраиваются в разделе [Canvas](Canvas.md):

- цвет текста;
- цвет фона;
- цвет рамки;
- толщина рамки.


## Camera settings

Здесь хранится список всех камер сцены.

Для камер доступны действия:

- добавить новую камеру;
- копировать существующую;
- удалить камеру;
- настроить положение, направление, тип и область отображения.

Если камер несколько, панель может делиться на несколько окон просмотра.

Подробное описание параметров камеры:

- [PerspectiveCamera](Editors/Cameras/PerspectiveCamera.md)
- [OrthographicCamera](Editors/Cameras/OrthographicCamera.md)

## Lighting settings

Настройки освещения сцены.

### Default ambient light color

Цвет встроенного ambient light, который всегда присутствует в демо-сцене.

### Default ambient light intensity

Яркость встроенного ambient light. Уменьшайте значение, если материалы модели выглядят слишком светлыми.

Список всех дополнительных источников света сцены.

Для каждого источника можно:

- выбрать тип;
- настроить яркость, цвет и положение;
- включить тени, если тип света это поддерживает;
- добавить вспомогательный визуальный маркер;
- копировать и удалять источник.

Подробности по каждому типу света:

- [AmbientLight](Editors/Lights/AmbientLight.md)
- [DirectionalLight](Editors/Lights/DirectionalLight.md)
- [HemisphereLight](Editors/Lights/HemisphereLight.md)
- [PointLight](Editors/Lights/PointLight.md)
- [RectAreaLight](Editors/Lights/RectAreaLight.md)
- [SpotLight](Editors/Lights/SpotLight.md)

## Global text settings

Это общие настройки всплывающих текстовых окон элементов.

Они применяются к тем объектам, у которых в локальных настройках текста включён параметр **Use global text settings**.

Если у элемента глобальные настройки отключены, он использует свои собственные параметры текста.

### Font size (px)

Размер шрифта в пикселях.

### Text color

Основной цвет текста.

### Max window width (px)

Максимальная ширина текстового окна. Полезно, чтобы длинный текст не растягивал окно слишком широко.

### Use real visual center for text position

Если включено, текст позиционируется относительно реального визуального центра объекта. Это удобно для сложных моделей и нестандартных форм, где обычный центр координат может выглядеть неестественно.

### Show text statically (no hover required)

Текст показывается постоянно, без наведения.

### Show text if data condition match

Если постоянный показ текста выключен, этот параметр позволяет показывать текст только при выполнении условия по данным.

### Show leader line

Добавляет линию от объекта к текстовому окну.

Дополнительные параметры:

- **Custom leader line color** — использовать отдельный цвет линии;
- **Leader line color** — цвет линии;
- **Leader line width (px)** — толщина линии.

### Background color

Фоновый цвет текстового окна по умолчанию.

### Border color

Цвет рамки текстового окна.

### Border width (px)

Толщина рамки.

Полная локальная настройка текстовых окон описана в [TextSettings.md](TextSettings.md).

## Help-info panel position

Определяет, в каком углу панели будет появляться вспомогательная информационная панель при наведении на элемент, если у этого элемента включено **Add auxiliary information**.

## Help info size

Размер вспомогательной информационной панели. Обычно используется как ограничение по ширине.

## Add gradient

Включает общую цветовую шкалу панели.

Она используется там, где элемент или текстовое окно должны окрашиваться по данным через общий градиент.

## Gradient settings

Параметры общей шкалы цвета.

### Основные параметры

- **Lower gradient boundary color** — цвет нижней границы шкалы;
- **Upper gradient boundary color** — цвет верхней границы шкалы;
- **Lower gradient boundary value** — минимальное значение диапазона;
- **Upper gradient boundary value** — максимальное значение диапазона.

### Gradient type

Способ расчёта градиента. Подходит для разных задач интерполяции цвета по точкам.

В зависимости от выбранного метода могут появляться дополнительные параметры, например:

- диапазон вариограммы;
- nugget-эффект;
- plateau;
- параметр формы для RBF;
- сглаживание;
- тип базисной функции.

### Show legend on scene

Показывает легенду градиента прямо поверх сцены.

Дополнительные параметры легенды:

- **Legend position** — положение в панели;
- **Legend title** — подпись шкалы;
- **Legend width (px)** — ширина легенды;
- **Legend bar height (px)** — высота цветовой полосы;
- **Legend steps (tick count)** — количество делений.

## Практический совет

Если вы настраиваете панель впервые, начните в таком порядке:

1. Добавьте элементы в **Elements Configuration**.
2. Настройте нужную камеру в **Camera settings**.
3. Выставьте освещение в **Lighting settings**.
4. Добавьте подписи и всплывающий текст.
5. Уже после этого включайте общую цветовую шкалу, анимации и вспомогательные панели.
